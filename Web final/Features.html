키워드 기반 취업 정보 웹 크롤링 시스템

복잡하고 방대한 취업 자료를 한곳에.

키워드를 입력하여 검색하면, 취업 정보 사이트(indeed, 사람인)에서
입력한 키워드를 바탕으로 취업정보를 모아 한 눈에 볼 수 있는 웹 사이트 입니다.

1.

취업 정보 사이트에서 키워드에 맞춰 검색된 결과 페이지를
웹 크롤링 라이브러리인 requests와Beautifulsoup를 이용하여 크롤링 합니다.

2.

크롤링을 통해 모은 데이터를
표 형식으로 결과창에서
보여줍니다.

3.

사용자는 표에 있는 지원하기
링크를 눌러 해당 구직 페이지에 리다이렉트 하여 정보를
상세하게 확인할 수 있습니다.

4.

이 외에도,
datetime 라이브러리를
활용하여 사용자가 입력한
키워드와 현재시각이 포함된
제목으로 저장된 결과파일을
받아 볼 수 있습니다.
